Keyboard Shortcuts

Grab or Release Balloon

Space or Enter keys grab or release balloon.

Jump Grabbed Balloon

  • J plus S Jumps close to Sweater.
  • J plus W Jumps to Wall.
  • J plus N Jumps to Near wall.
  • J plus C Jumps to Center.

Move Grabbed Balloon

  • Arrow keys move grabbed balloon.
  • W, A, S, or D keys move grabbed balloon up, left, down, or right.
  • Hold down Shift key and press a move key to move in smaller steps.

General Navigation

  • Tab key moves to next item.
  • Shift plus Tab key moves to previous item.
  • Escape key exits a dialog, like this one.
